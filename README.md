# Дипломная работа

Состоит из трёх частей — серверная разработка, верстка и JS.
В каждой папке есть `review.md` с комментарием от код-ревьюера. 

### Backend
Что было реализовано: 
* созданы схемы и модели user и article;
* созданы роуты `GET /users/me`, `GET /articles`, `POST /articles` и `DELETE /articles/articleId`;
* реализована создание и авторизация пользователей через роуты `POST /signup` и `POST /signin`;
* возвращение JWT при успешной авторизации;
* настроено логгирование реквестов и ошибок;
* валидация полей запросов на серверной стороне;
* подключена MongoDB для хранения данных Users и Articles;

_следующие этапы были реализованы на момент сдачи диплома_
* создана виртуальная машина на Ubuntu 20.04 в Yandex Cloud;
* создан домен на `my.freenom.com`;
* настроен Nginx;
* настроен SSL-сертификат;
* настроена работа API на виртуальной машине, доступ к запросам осуществлён по адресу `api.newsexplore.tk`;
* перезапуск приложения (в случае сбоя) при помощи `PM2`.

### FrontEnd
Что было реализовано:
* адаптивная вёрстка двух страниц;
* настроен Webpack, локальный сервер с Hot-reload, Babel и PostCSS для сборки проекта;
* семантика и БЭМ при вёрстке (кроме файловой структуры по БЭМ);
* использование flex и grid layout;
* валидация полей со стороны браузера.

### JS 
Что было реализовано:
* открытие/закрытие попапа регистрации/авторизации, передача данных на сервер при регистрации;
* работа с NewsAPI, поиск новостей по ключевым словам, отображение найденных статей;
* изменение шапки сайта по состоянию авторизации клиента;
* сохранение статей, передача в базу данных;
* отображение прелоудера, плейсхолдер при отсутствии новостей по ключевым словам;
* вывод сохраненных карточек на странице пользователя;
* использование ES6 классов и модулей, Fetch для работы с API.
