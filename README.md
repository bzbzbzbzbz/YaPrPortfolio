# Практическая работа №11

Возвращение к девятой практической работе.

Работа с node.js, Webpack, Babel и ES6-модулями. Деплой бандла на GitHub Pages. 

### Инструкция по применению:
1. Скопируйте репозиторий и через терминал введите `npm install`; 
2. Через терминал введите `npm run build` для Build сборки;
3. Через терминал введите `npm run dev` для Dev сборки и открытия страницы на локальном сервере с hot-reload;
4. Через терминал введите `npm run deploy` для загрузки Build сборки на GH-pages;

_Все те же ограничения по API, что и у девятой работы._
