# Практические работы №12-15

Работа с express.js, MongoDB. Создание API, деплой Backend'а.

### 12 практическая работа
Что было реализовано:
* настроен editorconfig;
* настроен eslint;
* запущен сервер на express.js;
* `npm run start` запускает сервер на localhost:3000l;
* `npm run dev` запускает сервер на localhost:3000 c hot-reload;
* при переходе на `localhost:3000` открывается FrontEnd девятой работы; 
* настроен роут `GET localhost:3000/users` - возвращает JSON-объект из файла `users.json`;
* настроен роут `GET localhost:3000/cards` - возвращает JSON-объект из файла `cards.json`;
* настроен роут `GET localhost:3000/users/8340d0ec33270a25f2413b69` - возвращает JSON-объект из файла `users.json` c ID пользователя;
* возврат 404 при некорректном ID или некорректном запросе.

### 13 практическая работа
Что было реализовано:
* настроена и подключена MongoDB;
* добавлены схемы User и Card;
* добавлен роут и контроллер `POST localhost:3000/users` - добавление пользователя в БД;
* добавлены роуты и контроллеры `POST localhost:3000/cards` и `DELETE localhost:3000/cards/:cardID` - добавление и удаление карточки в БД.
